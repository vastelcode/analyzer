# Приложение Analyzer

## Введение

Не трудно догадаться, что данное приложение предназначено для автоматизации одной из рутинных задач - анализа данных.
Используя Analyzer, вы можете обрабатывать большие объёмы данных на предмет нахождения соответствия регулярным выражениям, выбранных вами.

Вам, как пользователю, предоставлено три варианта взаимодействия с программой:

- CLI (консольный интерфейс) отлично подойдёт для любителей Linux, абсолютно всё в консоли: операции, ввод, вывод, результаты
- GUI (графический интерфейс), программа открывается в отдельном окне, где вас встречает панель управления и поле вывода
- WEB (веб-интерфейс), на вашем ПК разворачивается локальный сервер, открывается сайт в окне браузера и вы видите дашбоард

Стоит отметить, что точкой входа в любом случае будет консоль, а далее уже выбирайте, что вам по душе.

Давайте перечислю вам функционал Analyzer-а:

- глубокий/поверхностный анализ указанной директории на вашем ПК(в глубоком режиме проверяются ВСЕ подпапки, находящиеся в корневом каталоге)
- глубокий/поверхностный парсинг веб-страниц (в глубоком режиме будут проверяться всевозможные части одного сайта)
- отображение статистики по операциям
- возможность создания нескольких профилей для одного ПК

На этом с введением всё, теперь можете начать работу с приложением по следующему алгоритму:

- открыть терминал
- выбрать директорию, где у вас располагается программа Analyzer
- запустить файл index.py(точка входа)
